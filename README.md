# Abyss

## User Services
- `wlsunset`: Create a conf file in the service directory `~/.service/wlsunset`
              Customize lat and long with latitude and longitude (otherwise gets it from ipinfo.io/loc).
              Can if set and rise are set uses time instead of coordinates. Change low and high for the temperature control.
              Can set gamma with gamma
              
## User groups
    It is required the user is in the \_seatd group as this build uses seatd to start niri
